# Калькулятор

## 1. Описание проекта

Простой веб-калькулятор с графическим интерфейсом. Поддерживает базовые арифметические операции, а также обрабатывает ошибки вычислений.

## 2. Функциональность

### Поддерживаемые операции
- Сложение
- Вычитание
- Умножение
- Деление
- Десятичные дроби
- Очистка поля

### Ограничения
- Не поддерживаются скобки
- Оператор не может быть последним символом
- Не поддерживаются отрицательные числа в начале выражения
- Нельзя вводить несколько точек в одном числе
- Нельзя вводить два оператора подряд

### Обработка ошибок
- Деление на ноль выводит "Ошибка"
- Некорректное выражение не вычисляется
- При ошибке калькулятор очищается и готов к новому вводу

### Класс Calculator
В нём хранятся:
- `display` — экран калькулятора
- `expression` — данные, введённые пользователем
- `hasError` — была ли ошибка

И методы:
- `init()` — запуск калькулятора
- `clear()` — очищает
- `handleButtonClick()` — обрабатывает нажатия
- `calculate()` — считает результат
- `updateDisplay()` — обновление экрана

## 4. Логика работы

### Как работает
1. Нажимаю кнопку с цифрой
2. Цифра добавляется в выражение
3. При нажатии `=` вычисляется результат
4. Результат выводится на экран

### Валидация
Перед добавлением символа проверяю:
- Запрет оператора в начале выражения
- Не начинается ли выражение с `*` или `/`
- Нет ли лишней точки в данном числе

### Вычисление
Перед вычислением проверить выражение на правильность. Если всё хорошо, то считает. Если ошибка, то выводится сообщение об ошибке.
